# Submission Bundle (Milestone 2 + Milestone 3)

This folder contains two ready-to-review packages:

- `milestone2/` -> copy of `submission_m2/`
- `milestone3/` -> copy of `submission_m3/`

Combined guidance (what to use from M2 and what is required in final M3 ZIP):
- `COMBINED_M2_M3_SUBMISSION.md`

## What should be in Milestone 3 ZIP (required by assignment)

Inside `milestone3/`, final zip should include:
- `paper.pdf`
- `demo_video.mp4` or `demo_link.txt`
- `code/`
- `results/`
- `README.md`
- `requirements.txt`

## Build Milestone 3 ZIP

From `milestone3/`:

```bash
./make_zip.sh <LastName> <FirstName>
```

This creates:
`CS5590_Grad_M3_<LastName>_<FirstName>.zip`

## Note

If `paper.pdf` and demo file are not present yet, add them to `milestone3/` first.
